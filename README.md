## Deskripsi Aplikasi

Aplikasi ensiklopedia digital untuk mengeksplorasi flora dan fauna endemik, yang mengintegrasikan data API dengan penyimpanan offline (Room) untuk akses informasi yang cepat dan efisien.

## Fitur yang Diimplementasikan

- Sinkronisasi Data: Pengambilan data dari API dan penyimpanan otomatis ke database lokal (Room).
- Kategorisasi Fragment: Pemisahan otomatis data Hewan dan Tumbuhan dengan layout mandiri.
- Navigasi Bottom Bar: Perpindahan antar kategori menggunakan menu bawah dengan ikon kustom.
- Pencarian & Favorit: Fitur cari data secara real-time dan simpan data ke daftar favorit.
- Profil Mahasiswa: Pop-up Bottom Sheet berisi foto dan identitas pembuat aplikasi.
- Mode Gelap (Dark Mode): Dukungan tema gelap dengan ikon yang menyesuaikan warna secara otomatis.
